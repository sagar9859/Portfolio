Tune Sphere

Tune Sphere Music Web Application is a project developed during the AICTE internship offered by Edunet Foundation. It is a web-based music player that allows users to explore, search, and play their favorite music tracks. The application leverages HTML, CSS, and JavaScript

BE final Year Project

The Retrieving Indian Music by Querying with Humming project is a music retrieval system that enables users to search for Indian songs by humming a part of the melody. This project uses Python for processing and analyzing the hummed queries
